# Fake News Detection AI

AI powered fake news detection web application using NLP and Machine Learning.

## Features

* Detects fake and real news articles
* URL based article analysis
* Probability score visualization
* Authenticity score
* Professional responsive UI
* Machine Learning powered predictions

## Technologies Used

* Python
* Flask
* Scikit-learn
* TF-IDF Vectorization
* Logistic Regression
* HTML
* CSS

## How to Run

### Clone Repository

```bash
git clone https://github.com/grashma123/Fake-News-Detection-AI.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

### Open Browser

```text
http://127.0.0.1:5000/
```
